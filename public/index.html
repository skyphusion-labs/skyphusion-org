Flagship

Vivijure

A self-hosted AI film studio you own, nearing its full public release. Run the control panel on Cloudflare (vivijure-cf), on a home computer, or on any cloud server (vivijure-local); both hosts share vivijure-core. The Studio is a thin module host: storyboard, cast, render orchestration, and a typed contract that routes every stage through swappable backends. Rent high-end GPU time by the second, run motion on your own card, call cloud i2v endpoints per shot, and keep assembly on cheap CPU containers you host (vivijure-cf/containers). Meet it at vivijure.com or try the live demo.

CPU media stack (your iron, not your GPU bill)

Five always-on Docker services in vivijure-cf/containers (also used by the self-hosted host). On Cloudflare they are reached over Workers VPC; the host presigns object-store URLs and the containers stay stateless and credential-free. GPU money goes to diffusion; concat, mux, captions, portrait prep, beat sync, and loudness work run on CPU.

Infrastructure

Postern

Email for humans and agents. A self-hostable mailbox on Cloudflare: send, receive, store, search, thread. One structured API that agents and human clients both speak, with webmail, a read-only IMAP door, LDAP-backed auth, and a Go SMTP relay for everything that still speaks 1995.

Mailbox AGPL

Postern

Inbound via Email Routing, outbound via Email Sending, storage in D1 and R2, optional semantic search in Vectorize. Webmail and IMAP are read-only doors onto the same store; sending stays on the structured API so nothing drifts.

github.com/skyphusion-labs/postern
IMAP Production

The IMAP door

Server-side SEARCH pushdown, ENVELOPE fidelity v2, RFC 2177 IDLE, LDAP group-gated auth on 993, trust-gated PROXY protocol behind the load balancer. Built by pointing real clients at it and fixing what they choked on.

Agents MCP

Agents welcome

Scope-gated MCP send tools, a per-identity send registry, MTA-STS policy route, and a Go relay that bridges SMTP-only callers. The mailbox the whole stack is moving toward.

Recent milestones
IMAP grew from demo to production door: SEARCH pushdown, envelope fidelity, LDAP on both submission and IMAP doors, containerized relay and IMAP images on GHCR, per-identity send registry, and env-gated MTA-STS. Read the write-up on skyphusion.net.

Operating principles

How we work

  1. 01
    Not for sale. Free forever. Everything here is open source and not for sale. Made with love, developed in the open. Tools should be bought or built, not rented.
  2. 02
    Own it over rent it. Your GPU, your keys, your data. We build for sovereignty, not convenience fees.
  3. 03
    Verify, do not assume. Punk DIY ethos with an aviation-grade finish. Release gates render a real film before they promote. Measure twice.
  4. 04
    AGPL by default. If you run it as a network service, you owe your users the source. Keeps free things free.
  5. 05
    Edge-native, not edge-washed. Cloudflare Workers are not a deployment target; they are the architecture.
  6. 06
    Kill the cop in your head. Permission was never required. Fork it. Break it. Make it yours.
